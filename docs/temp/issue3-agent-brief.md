### Goal

Initialize the Node.js/TypeScript project and create the SQLite database schema that underpins the entire system. Write tests first (TDD), then implement to make them pass.

### Design Decisions (from grilling)

- **Schema init:** Simple `CREATE TABLE IF NOT EXISTS` in a single module called at app startup — no migration runner framework.
- **Production DB:** `./data/mtgnews.db` with env var override (`MTGDB_PATH`).
- **Test DB:** `:memory:` in-memory SQLite per test for speed and isolation.
- **`signals` table:** Includes both `created_at` (INTEGER NOT NULL, when Signal row inserted) and `processed_at` (INTEGER, nullable, set when LLM analysis completes).
