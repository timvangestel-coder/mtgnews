Architecture Context

The view rendering seam between EJS templates and Alpine.js has a critical module:

  • signal-detail.ejs defines window.__signalDetail() in an inline blocks.

    This means the window.__signalDetail = function() {...} definition stays inside <%- body %>, embedded in the rendered HTML. It never gets hoisted to the scripts variable that layout.ejs renders before Alpine.js.
